# 📘 Lecture Notes Chatbot

A Python chatbot that summarizes lecture notes and answers follow-up questions using AI.

---

## 🚀 Features
- Generates summary with title, key points, and action items
- Supports follow-up questions
- Remembers previous conversation (multi-turn memory)
- Handles unrelated questions politely
- Error handling included

---

## 🛠 Setup

1. Install Python  
2. Install required libraries:
   pip install openai python-dotenv

3. Create a .env file and add:
   OPENAI_API_KEY=your_api_key_here

---

## ▶️ How to Run

Run the chatbot using:

python chatbot2.py

---

## 💬 Usage

1. Paste your lecture notes  
2. Get a clean summary  
3. Ask follow-up questions  

---

## ⚠️ Note
- Do not share your API key
- .env file is not uploaded for security
